# Task Manager Full Stack App

This project contains a beginner-friendly Task Manager application with separate `backend` and `frontend` folders.

## Backend

1. Open `backend` folder.
2. Run `npm install`.
   

### Run
- `npm start` 
- `npm runev` 
- `npm run dev` 


### API Endpoints
- `GET /api/tasks`
- `GET /api/tasks/:id`
- `POST /api/tasks`
- `PUT /api/tasks/:id`
- `DELETE /api/tasks/:id`

## Frontend

### Setup
1. Open `frontend` folder.
2. Run `npm install`.

### Run
- `npm run dev`
